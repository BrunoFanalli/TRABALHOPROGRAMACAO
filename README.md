Sistema Acadêmico - Trabalho Final de Programação
📌 Sobre o Projeto

Este projeto consiste em um sistema acadêmico desenvolvido como trabalho final da disciplina de programação. O objetivo é gerenciar usuários (alunos) com informações como nome, CPF, data de nascimento e senha, além de permitir a visualização e administração desses dados por um administrador.

O sistema simula um ambiente escolar onde há controle básico de cadastro e gerenciamento de usuários.

🎯 Funcionalidades
👤 Usuário
Cadastro com:
Nome
CPF
Data de nascimento
Senha
Login no sistema
Visualização de seus dados
🔐 Administrador
Visualizar todos os usuários cadastrados
Adicionar novos usuários
Remover usuários existentes
Acesso aos dados:
Nome
CPF
Senha cadastrada

⚠️ O administrador não pode lançar notas, apenas gerenciar usuários.
